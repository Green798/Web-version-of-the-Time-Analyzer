# 时间分析器网页版

这是一个简单的网页版时间分析器应用，用于记录和分析每天在不同时间段做的事情。

## 功能特点

- **时间记录**：记录每天不同时间段的活动
- **活动管理**：自定义添加不同类型的活动
- **数据可视化**：通过图表直观展示时间分布
- **数据统计**：统计不同活动的时间占比
- **历史查询**：查看历史日期的时间记录

## 使用方法

### 主页

1. 选择当前正在进行的活动
2. 点击"开始记录"按钮开始记录时间
3. 活动结束后点击"结束记录"按钮
4. 可以通过"添加新活动"按钮添加自定义活动

### 分析页面

1. 通过日期选择器选择要查看的日期
2. 查看24小时活动分布图
3. 查看活动时间占比统计
4. 查看详细的时间记录列表

## 技术实现

- **前端**：HTML5, CSS3, JavaScript
- **UI框架**：Bootstrap 5
- **图表库**：Chart.js
- **数据存储**：浏览器 LocalStorage

## 本地运行

1. 克隆或下载本项目到本地
2. 使用现代浏览器打开 `index.html` 文件
3. 也可以使用任何静态文件服务器托管项目文件
